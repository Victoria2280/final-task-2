# Final Task 2 — Shop Database

A console-based application developed as the second final project for the Programming Course at KIT (Karlsruhe Institute of Technology).

## About

A small database system for a shop, implemented in Java. Allows managing shop data through the console.

## Technologies

- Java
- IntelliJ IDEA

## How to Run

1. Clone the repository:
```bash
   git clone https://github.com/Victoria2280/final-task-2.git
```
2. Open the project in IntelliJ IDEA
3. Run the main class from `src/edu/kit/kastel/`
